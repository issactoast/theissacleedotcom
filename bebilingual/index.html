R to Python 코드 변환사전

이 강의는 저의 개인적인 코드 모음이라고 생각하셔도 될 것 같습니다.

최대한 간략하게

이 강의는 저의 다른 강의와는 다르게 최대한 간략하게 적어 참고형식이 되도록 하는 것이 목적입니다. 따라서 두 가지 언어 중 하나는 확실하게 알고 계신분께 도움이 될 것 같습니다.

왜 하시나요?

데이터 사이언티스트 중 R과 Python 두 개를 모두 쓰시는 분들을 위한 사전 작성.

마치며

Anyone can learn anything. 저의 홈페이지 문구이기도 하고, 저의 모토이기도 한 말입니다. 프로그래밍은 하나의 언어 입니다. 언어는 쓰면 쓸쓰록 느는 것이죠! 나는 안되는구나 하는 생각보다는 계속 쓰면서 하나씩 정복해 나가 보세요. 건승을 기원합니다!
